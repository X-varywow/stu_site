# stu_site
这是一个高校学生社区之类的网站（完整项目），组织内容的方式有别于BBS。基础功能已实现：登录、注册、发表、评论、点赞、个人中心、基于分类板块的查看。

## 文件说明

- css
- favicons
- fonts
- hphotos，存放用户头像
- img，存放网站本身的图片
- js
- logic，主要的后端处理逻辑，都是 php 写的
  - ajax-info，内容展示的获取逻辑
- main, 主要场景，由于需要验证用户信息，使用 php 保存了
- userimg，存放用户存放的图片
- index.html，初始的登录界面
- register.html，注册界面
- stu_site.sql.gz，数据库的压缩包，可参考表的格式
